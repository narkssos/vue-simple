# vue-simple

> A Vue.js project

## 환경설정

### 1. NodeJS 설치
https://nodejs.org/en/download/ 에 방문하여 운영체제에 맞는 버전을 설치


###2. 소스 받기
``` bash
git clone https://github.com/narkssos/vue-simple.git
```

###3. 필요한 라이브러리 설치
``` bash
npm install
```

###4. 개발 모드 실행  at localhost:9999

``` bash
npm run dev
```

###5. 빌드시에 사용 -- 해당없을듯.
``` bash
npm run build
```
